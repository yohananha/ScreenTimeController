One more setting

ScreenTime needs "Display over other apps" permission so it can show the gentle reminder when time's up. We never collect what you watch — only when the limit hits.

Open Android settings
Why do you need this?
This permission lets ScreenTime draw a single overlay window — the block screen — when time's up. It can't see the content of any app, only when a limit is hit. You can revoke it in Android Settings → Apps → ScreenTime anytime.

Almost there!

Turn on the ScreenTime accessibility service so it can tell which app is open and step in only when needed. You can review what it has access to in Android Settings any time.

Open Android settings
What does this do?
The Accessibility Service watches for app-switch events — the same signal that tells ScreenTime "YouTube just opened." It cannot read text, see passwords, or interact with apps on your behalf. Disable it in Android Settings → Accessibility → ScreenTime whenever you like.
